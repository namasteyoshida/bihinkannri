# 備品貸出処理画面 設計書

## 1. 概要

### 画面名
備品貸出処理画面

### 目的
備品名・利用者名・貸出日を入力し、貸出記録として一覧表示する。

### 使用技術
- React
- TypeScript
- react-hook-form

---

# 2. コンポーネント概要

## コンポーネント名
`LendingForm`

## 役割
- 貸出情報入力フォームの表示
- 入力値のバリデーション
- 貸出記録の追加
- 貸出記録の削除

---

# 3. データ定義

## LendingFormValues（フォーム入力値）

| 項目 | 型 | 内容 |
|---|---|---|
| equipmentName | string | 備品名 |
| userName | string | 利用者名 |
| lendDate | string | 貸出日 |

## LendingRecord（貸出記録）

| 項目 | 型 | 内容 |
|---|---|---|
| id | number | 識別ID |
| equipmentName | string | 備品名 |
| userName | string | 利用者名 |
| lendDate | string | 貸出日 |

---

# 4. State管理

| State | 内容 |
|---|---|
| lendingList | 登録された貸出記録一覧 |
| maxId | 貸出記録に付与するID |

---

# 5. フォーム項目仕様

| 項目 | 入力形式 | バリデーション |
|---|---|---|
| 貸出備品 | テキスト | 必須 |
| 利用者名 | テキスト | 必須、30文字以内 |
| 貸出日 | 日付 | 必須 |

---

# 6. 処理仕様

## 貸出登録処理（onsubmit）

1. 入力内容を取得
2. 貸出記録を作成
3. lendingListへ追加
4. IDを1増加
5. フォームをリセット

---

## エラー処理（onerror）

入力エラー発生時にエラー内容をconsoleへ出力する。

---

## 貸出取消処理（handleRemove）

1. 取消ボタンのIDを取得
2. 対象データを除外
3. lendingListを更新

---

# 7. 画面構成

## 入力画面

- 貸出備品入力欄
- 利用者名入力欄
- 貸出日入力欄
- 貸出実行ボタン

## 貸出記録一覧

登録済みデータを以下の形式で表示する。
利用者名 さんが「備品名」を借りました
日付：YYYY-MM-DD
[取消]


---

# 8. 使用Hooks

| Hook | 用途 |
|---|---|
| useState | 貸出一覧・ID管理 |
| useForm | フォーム管理、入力チェック |
| useId | labelとinputの紐付け |

---

# 9. 処理フロー
ユーザー入力
↓
バリデーション
↓
貸出登録
↓
State更新
↓
貸出記録表示
